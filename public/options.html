Compact Configuration

Configure Compact and validate local Ollama connectivity.
Not required for standard Ollama setups.
Standard: http://localhost:11434/v1
e.g., llama3.2:1b, mistral, gemma
Resolves 403 errors during large repository scans by increasing GitHub API rate limits.

CORS Fix (Required)

If you get a 403 error, set this Environment Variable on your PC:

OLLAMA_ORIGINS = chrome-extension://*

Then Restart Ollama (Quit from tray and restart).
